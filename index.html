Mouse Press Counter - Demo

Set a custom classes and set min/max value:

$('.wrap-count-1').mousePressCounter({
    inc: '.number-inc',
    dec: '.number-dec',
    counter: '.number-input',
    min: 5,
    max: 100
});

Increment the number with a linear speed:

$('.wrap-count-2').mousePressCounter({
    timeStep: 0
});

Increasing speed every 5 iteration on 10ms:

$('.wrap-count-3').mousePressCounter({
    timeStep: 10
});
Default counter:
$('.wrap-count-4').mousePressCounter();
